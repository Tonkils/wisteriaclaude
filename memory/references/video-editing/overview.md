## What's Here

This directory contains a comprehensive video editing reference library covering three domains, plus cross-cutting resources.

## Decision Tree — Which File to Load

**Is the task about a specific editing domain?**
- Editing a podcast (audio or video) → `podcast-editing.md`
- Editing Instagram Reels, TikTok, or YouTube Shorts → `shortform-editing.md`
- Editing a YouTube video (long-form) → `youtube-editing.md`

**Is the task about a cross-cutting topic?**
- Audio processing, EQ, compression, loudness → `audio-engineering.md`
- Choosing or comparing editing software → `tools-and-software.md`
- Finding tutorials, courses, or learning paths → `tutorials-and-courses.md`
- Studying what top creators do and why → `creators-and-benchmarks.md`

**Load multiple files when:**
- Creating podcast clips for social media → `podcast-editing.md` + `shortform-editing.md`
- Setting up an audio processing chain → domain file + `audio-engineering.md`
- Starting from scratch and choosing tools → `tools-and-software.md` + the relevant domain file

## Universal Principles (Apply to All Editing)

1. **Story first, technique second.** Every edit serves the narrative. If a cut doesn't improve the story, don't make it.
2. **Audio is 50% of the experience.** Bad audio ruins good video. Process audio before worrying about visuals.
3. **The hook determines everything.** The first 1-5 seconds decide whether anyone watches the rest. Edit the hook last, when you know the best moment.
4. **Cut for the audience, not the creator.** Remove ego, keep engagement. If a segment is self-indulgent but boring, cut it.
5. **Specs are not negotiable.** Loudness targets, aspect ratios, and format requirements exist for a reason. Hit them exactly.
6. **Watch on the delivery device.** Always preview edits on the device/platform where the audience will consume them — phone for Reels, laptop for YouTube, headphones for podcasts.
